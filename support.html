Wie können wir helfen?

Antworten auf häufige Fragen und direkter Kontakt zu unserem Support-Team.

Häufige Fragen

Wie importiere ich Produktstammdaten aus Excel?

Öffnen Sie die App und tippen Sie auf „Stammdaten importieren". Wählen Sie Ihre Excel-Datei aus. Die Spalten für EAN, Artikelnummer und Produktname werden automatisch erkannt. Sie können die Zuordnung vor dem Import noch anpassen.

Welche Barcodes werden unterstützt?

StockScan unterstützt EAN-13, EAN-8, UPC-A und Code 128. Die Erkennung erfolgt automatisch beim Scannen – Sie müssen den Typ nicht manuell auswählen.

Wie funktionieren mathematische Ausdrücke bei der Mengeneingabe?

Geben Sie bei der Menge einen Ausdruck wie "12*5" ein – StockScan berechnet das Ergebnis automatisch (60 Stück). So können Sie z.B. Kartons mit Stückzahl direkt erfassen.

Was passiert bei doppelt gescannten Artikeln?

StockScan erkennt Duplikate automatisch und fragt Sie, ob die neue Menge zur bestehenden addiert oder ob die Erfassung ersetzt werden soll.

Wie exportiere ich die Inventurliste?

Öffnen Sie die gewünschte Inventurliste und tippen Sie auf das Export-Symbol oben rechts. Wählen Sie „Als Excel exportieren" – die XLSX-Datei wird erstellt und Sie können sie direkt teilen.

Werden meine Daten in der Cloud gespeichert?

Nein. Alle Daten bleiben ausschließlich auf Ihrem Gerät. StockScan benötigt keine Internetverbindung und sendet keine Daten an externe Server.

Kann ich StockScan auf mehreren Geräten nutzen?

Jedes Gerät speichert seine eigenen Daten lokal. Eine Synchronisierung zwischen Geräten ist derzeit nicht vorgesehen. Exportierte XLSX-Dateien können Sie manuell auf andere Geräte übertragen.

Die Kamera erkennt den Barcode nicht – was tun?

Stellen Sie sicher, dass der Barcode gut beleuchtet und nicht beschädigt ist. Alternativ können Sie die Artikelnummer manuell eingeben. Falls das Problem anhält, kontaktieren Sie uns per E-Mail.

Weitere Fragen?

Unser Support-Team hilft Ihnen gerne weiter. Schreiben Sie uns eine E-Mail und wir melden uns so schnell wie möglich.
