Funktionen

Alles was Sie für eine
professionelle Inventur brauchen

Von der Barcode-Erfassung bis zum fertigen Excel-Report – StockScan deckt Ihren gesamten Inventurprozess ab.

▦

Barcode-Scanner

EAN-13, EAN-8, UPC-A und Code 128 werden zuverlässig erkannt. Mit intelligenter Duplikaterkennung und manueller Eingabe als Fallback.

📊

Stammdaten aus Excel

Importieren Sie Ihre bestehenden Produktdaten direkt aus Excel. Automatische Erkennung von Produktname, EAN und Artikelnummer.

🧮

Flexible Mengenerfassung

Mathematische Ausdrücke wie "12*5" werden direkt berechnet. Individuelle Einheiten pro Artikel und Kommentarfunktion inklusive.

📋

Mehrere Inventurlisten

Verwalten Sie verschiedene Zählungen parallel. Mit Farbcodierung und Icons behalten Sie stets die Übersicht.

📤

Excel-Export (XLSX)

Vollständiger Export aller Inventurdaten mit Zeitstempel. Einfaches Teilen per Mail, AirDrop oder Cloud-Dienst.

🔒

Datenschutz by Design

Alle Daten verbleiben ausschließlich auf Ihrem Gerät. Keine Cloud, keine Server, keine Registrierung erforderlich.

So funktioniert's

In 4 Schritten zur fertigen Inventur

Keine Einarbeitung, kein Setup – starten Sie in unter einer Minute.

01

Stammdaten importieren

Laden Sie Ihre Excel-Tabelle mit Produktdaten. Dieser Schritt ist optional – Sie können auch direkt loslegen.

02

Barcode scannen

Richten Sie die Kamera auf den Barcode oder geben Sie die Artikelnummer manuell ein.

03

Menge erfassen

Zählung eingeben – auch mit Rechenformeln wie "3*12". Kommentare für Sonderinformationen möglich.

04

Als Excel exportieren

Ein Tap genügt für den vollständigen XLSX-Export mit allen Details und Zeitstempeln.

Ihre Daten gehören Ihnen – und nur Ihnen.

In einer Zeit, in der jede App Ihre Daten in die Cloud sendet, geht StockScan einen anderen Weg. Vollständige Datensouveränität ohne Kompromisse.

  • Alle Daten bleiben auf Ihrem iPhone / iPad
  • Keine Internetverbindung erforderlich
  • Keine Benutzerkonten oder Registrierung
  • SwiftData für sichere lokale Speicherung
  • Kein Tracking, keine Werbung
🔐

Datenschutz-Versprechen

StockScan überträgt keine Daten an externe Server. Alle Inventurdaten, Produktstammdaten und Einstellungen werden ausschließlich lokal auf Ihrem Gerät gespeichert und verlassen dieses nur, wenn Sie aktiv einen Export durchführen.
